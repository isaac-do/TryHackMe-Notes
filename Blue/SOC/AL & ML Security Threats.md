- **Prompt Injection:** occurs when the original instructions provided to the model are overridden, often for malicious purposes like disclosing more information than it should, or generating harmful content.
- **Data Poisoning:** an attacker manipulates the training data /corpus used to train an AI model so its generated output is incorrect or biased.
- **Model Theft:** occurs when an attacker gains unauthorized access to an AI model. An attacker could potentially steal intellectual property.
- **Privacy Leakage:** the possibility that the AI model inadvertently revealing sensitive information about the data it was trained on, even if the data was supposed to be kept confidential.
- **Model Drift:** the potential for a model's performance to drift over time due to changes in the data or environment surrounding it.
